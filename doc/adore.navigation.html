<static> navigatePaths

Navigates the on-screen display to another path.
Parameters:
Name Type Description
offset number The offset of the current path from the desired path. E.g., an offset of 1 means the next path, an offset of -1 means the previous path.
onCompletion function Function to execute on completion of the path switching animation.
Source:
  • adore.navigation.js, line 18
Example
            navigatePaths(1);  // Next path.
            navigatePaths(-1); // Previous path.
            navigatePaths(2);  // The next but one path.
